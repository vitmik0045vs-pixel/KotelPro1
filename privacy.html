Политика конфиденциальности

Приложение "КотёлПро"
Последнее обновление: 05.02.2026

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей приложения "КотёлПро" (далее — Приложение).

Разработчик приложения "КотёлПро" (далее — Разработчик) уважает право пользователей на конфиденциальность и обязуется защищать персональные данные в соответствии с законодательством Российской Федерации.

2. Сбор персональных данных

Приложение "КотёлПро" НЕ СОБИРАЕТ, НЕ ХРАНИТ и НЕ ПЕРЕДАЁТ персональные данные пользователей третьим лицам.

Приложение работает в оффлайн-режиме и не требует подключения к интернету для выполнения расчётов.

3. Данные, используемые приложением

Приложение сохраняет только следующие данные, необходимые для работы пробного периода:

  • Дата первого запуска приложения — для расчёта оставшегося времени пробного периода;
  • Статус покупки — для определения, приобретено ли приложение.

Все указанные данные хранятся локально на устройстве пользователя и не передаются на серверы Разработчика или третьим лицам.

4. Использование данных

Собранные данные используются исключительно для следующих целей:

  • Предоставления 3-дневного пробного периода;
  • Определения статуса лицензии приложения (пробный период / куплено).

5. Защита данных

Разработчик принимает все необходимые меры для защиты данных пользователей от несанкционированного доступа, изменения, раскрытия или уничтожения.

Все данные шифруются и хранятся в защищённом хранилище устройства.

6. Права пользователя

Пользователь имеет право:

  • В любое время удалить приложение с устройства, что приведёт к удалению всех сохранённых данных;
  • Отказаться от использования приложения в любой момент.

7. Сторонние сервисы и реклама

Приложение "КотёлПро" НЕ СОДЕРЖИТ рекламы и НЕ ИСПОЛЬЗУЕТ сторонние аналитические сервисы (например, Google Analytics, Firebase и т.д.).

Приложение не содержит ссылок на сторонние веб-сайты или сервисы.

8. Изменения в политике конфиденциальности

Разработчик оставляет за собой право вносить изменения в настоящую Политику конфиденциальности в любое время.

Все изменения вступают в силу с момента их публикации в данном документе.

Пользователям рекомендуется периодически проверять данную страницу на наличие изменений.
